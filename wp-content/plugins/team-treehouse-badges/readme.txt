== Description ==

Pulls in your badges from Team Treehouse. Simply install add find the widget under appearance. Add a title (optional) and fill in your Treehouse username which is found at the end of your learning profile URL.

http://teamtreehouse.com/**psyne**

== Installation ==

If installing via the wordpress admin from the repository:
1. Activate the plugin through the 'Plugins' menu in WordPress

If manually uploading:
1. Download plugin archive in zip or gzipped tar format and extract the files on your computer.

1. Use an FTP or SFTP client to upload the trashemptier directory to wp-content/plugins directory of your WordPress installation. 

1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= How do I add the widget to my site =

Install the plugin and find it on the widget page under the appearance menu.

= Are there any CSS classes to style the widget =

The badges are contained with a parent with the id of badges_holder and a UL with the id of badges.
